# Barbershop App

Aplicación web para gestión de citas en peluquerías.

## Funcionalidades
- Reserva de citas online por parte del cliente
- Panel del barbero con calendario y gestión de citas
- Notificaciones por email al barbero y al cliente
- Confirmación de citas desde el email o el dashboard
- Autenticación con JWT para clientes y barberos

## Stack
- **Frontend:** React + Vite + Tailwind CSS
- **Backend:** FastAPI + SQLAlchemy
- **Base de datos:** PostgreSQL
- **Infraestructura:** Nginx + Docker Compose
